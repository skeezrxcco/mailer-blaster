# Goals

## MVP Feb - Mar 2026

- [ ] The system AI should be able to create a newsletter from a prompt
    It will take a single prompt and generate:
      - [ ] A proposition for the complete newsletter content and structure.
      - [ ] The user will be able to review, refine and approve the proposition.
      - [ ] The user will be able to upload images to the newsletter.
      - [ ] The user will be able to add links to the newsletter.
      - [ ] The user will be able to add recipients and the AI will be able to validate them.
      - [ ] The user will be able to manually send the newsletter or schedule it to be sent at a specific date and time.
      - [ ] The user will be able to see the status and get analytics of the newsletter (sent, scheduled, etc.).
      - [ ] The AI should be able to handle redonant inputs and generate a new proposition if needed.
      - [ ] The AI should be able to handle errors and provide feedback to the user.
      - [ ] The AI should be able to handle rate limits and provide feedback to the user.
      - [ ] The AI should be able to handle concurrent requests and provide feedback to the user.
      - [ ] The AI should be able to handle retries and provide feedback to the user.
      - [ ] The AI should be able to handle timeouts and provide feedback to the user.
      - [ ] The AI should be able to handle network errors and provide feedback to the user.
      - [ ] The AI should be able to handle invalid inputs and provide feedback to the user.
      - [ ] The AI should be able to handle invalid recipients and provide feedback to the user.
      - [ ] The AI should be able to handle invalid links and provide feedback to the user.
      - [ ] The AI should be able to handle invalid images and provide feedback to the user.
